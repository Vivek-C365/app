# Screens

This directory contains all screen components for the Animal Rescue mobile app.

## Available Screens

### ReportScreen
- Allows users to report animals in need
- Form with animal type, condition, location, and contact info
- Placeholder buttons for GPS and camera (to be implemented)
- Uses reusable Input, Button, and Card components

### CasesScreen
- Displays list of active rescue cases
- Shows case details including status, location, and reporter
- "I Can Help" button for volunteers to respond
- View details button for more information

### ProfileScreen
- User profile information and statistics
- Contact information display
- Notification preferences with toggle switches
- Edit profile and logout actions

## Component Usage

All screens use:
- Theme system for consistent styling
- Reusable components (Button, Card, Input, LoadingSpinner)
- React Native native components
- Proper error handling and loading states
